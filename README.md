# Mónica Montúfar - Portfolio Frontend

> "Especialista en Email Marketing y automatización."

Este proyecto es el front-end oficial para el portafolio online de Mónica Montúfar. Desarrollado con el enfoque arquitectónico vanguardista de "Astro Islands" para enviar *Zero JavaScript* por defecto, garantizando velocidades de carga espectaculares.

## Tecnologías Principales
- **Framework:** [Astro](https://astro.build)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Despliegue:** [Netlify](https://www.netlify.com/) (Manual Trigger)
- **Testing:** Playwright (E2E), Vitest (Unit), Lighthouse (a11y/Performance)

## Comandos Disponibles

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala las dependencias del proyecto |
| `npm run dev` | Levanta el servidor local (`localhost:4321`) |
| `npm run build` | Compila el sitio listo para producción en `./dist/` |
| `npm run preview` | Ejecuta el build localmente para revisión final |
| `npx playwright test` | Ejecuta los tests end-to-end |
