# Product Requirements Document (PRD)

## Visión General
El portafolio de **Mónica Montúfar (Món Mont)** es una carta de presentación interactiva, vibrante, y "premium" destinada a potenciar sus leads y su visibilidad como *Estratega de Ingresos y Arquitecta de Flujos de Automatización*.

## Objetivos Core
1. **Velocidad Extrema:** Alcanzar 98-100 en Lighthouse Score de Performance enviando Cero JavaScript base al cliente, lograble mediante Astro.
2. **Aesthetic UI:** UI construida con una vibrante paleta de colores sobre fondo oscuro/negro profundo. Acentos de cristal (Glassmorphism), sombras dramáticas, y bordes redondeados orgánicos.
3. **Seguridad y Legales:** Desglose impecable del banner de Cookies.
4. **Bilingüismo Silencioso:** Soporte nativo y SEO-perfecto de i18n para llegar al mercado angloparlante (URL base `/` para ES, `/en/` para EN).

## Secciones Requeridas (Homepage)
1. **Hero**: Imagen asimétrica, título, CTAs ('Portfolio', 'CV').
2. **Logos de Confianza**: (Kombutxa, Arval, Chokolate, etc.).
3. **Tarjeta Fortalezas**: Info personal ("Nómada actual: Colombia").
4. **Stack de Soluciones**: Grid dinámico (Entregabilidad, First-Party Data, etc.).
5. **Habilidades y Toolstack**: Progress bars 80-90% con íconos de software.
6. **Portfolio Highlight**: Casos de éxito y CTAs para expandir lectura.
7. **Testimonios**: Integración social minimalista.
8. **Blog Teaser**: Llamado a la acción "Visita mi Blog".
